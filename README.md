# StackOverflow Developer Analytics

Analytical SQL project querying a 200,000+ row dataset modelling
StackOverflow's post/answer/user schema. Built with DuckDB and Python.

## Overview

This project simulates a real data analytics workflow that generating a
realistic dataset, running 15 production-grade SQL queries, and
visualising the results with matplotlib.

**Dataset:** 200,000 questions · 350,000 answers · 50,000 users · 40 tags  
**Tools:** Python · DuckDB · pandas · matplotlib  
**Skills demonstrated:** Window functions · CTEs · aggregations ·
time-series analysis · funnel analysis · NTILE bucketing
